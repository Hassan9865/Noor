Noor-e-Islam - Complete Islamic Flutter App

🕌 Introduction

Noor-e-Islam is a comprehensive Islamic application built with Flutter using Stacked architecture. This MVP includes all essential features for Muslim users with a clean, modern UI and optimized performance.

✨ Key Features

🕋 Prayer Times

Accurate prayer times based on location with multiple calculation methods (Hanafi/Shafi)

📖 Complete Quran

Full Quran with Surah navigation, Ayah search, bookmarking, and direct page jump functionality

🧮 Zakat Calculator

Comprehensive calculator with gold/silver rate integration and detailed breakdown

🕋 Qibla Direction

Real-time compass showing accurate Qibla direction with device sensors

📿 Digital Tasbih

Interactive counter with vibration feedback and multiple counters

📚 Islamic Resources

Hadith collection, Daily Duas, 6 Kalmas with translations

🚀 Installation Guide

Follow these steps to set up the project:

1. Getting Started

  1. Download the package from CodeCanyon or your purchase source
  2. Extract the ZIP file to your preferred development directory

2. Project Setup

  1. Open the project folder in your preferred IDE
  2. Run the following command to get dependencies: 
    flutter pub get
  3. For iOS, navigate to the ios folder and run: 
    pod install
  4. Configure any necessary API keys in the configuration files

3. Running the App

  1. Connect a physical device or start an emulator/simulator
  2. Run the app using: 
    flutter run
  3. For release build: 
    flutter run --release
Pro Tip: If you encounter any issues during setup, try running flutter clean and then flutter pub get again. Also ensure your Flutter SDK is up to date.
